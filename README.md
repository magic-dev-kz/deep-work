# 🎯 Deep Work Timer

Track focused work sessions and build a deep-work habit.

**[Try it →](https://magic-dev-kz.github.io/deep-work/)**

## Features
- Customizable focus sessions with countdown timer
- Heatmap of your deep-work history
- Tags and session notes for context
- Insights dashboard — streaks, totals, trends
- Ambient focus sounds
- CSV export for your data

## Why Deep Work Timer?
Cal Newport proved that focused work drives results, but most timer apps stop at a Pomodoro beep. Deep Work Timer adds heatmaps, tags, and insights so you can see where your focus actually goes — and do more of what matters.

## Tech
- Single HTML file — nothing to install
- Works offline (PWA)
- No accounts, no tracking
- Your data stays in your browser (localStorage)
- Free forever
